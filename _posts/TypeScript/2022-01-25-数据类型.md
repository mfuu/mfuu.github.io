## TypeScript 基础类型
#### 1. Boolean 类型
```ts
let bool: boolean = false
// ES5: var bool = false
```

#### 2. Number 类型
```ts
let num: number = 9
// ES5: var num = 9
```

### 3. String 类型
```ts
let str: string = 'this is a string'
// ES5: var str = 'this is a string'
```

### 4. Array 类型
```ts
let arr: number[] = [1, 2]
// or
let arr: Array<number> = [1, 2]

// ES5: var arr = [1, 2]
```
